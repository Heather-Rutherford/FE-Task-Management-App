## Assignment: Task Management App with TypeScript

## Overview

In this TypeScript Task Management Application project, you'll embark on a journey to develop a robust task management tool using TypeScript and React. This project will enhance your understanding of TypeScript fundamentals while introducing you to the practical application of TypeScript in building real-world web applications.

## Requirements

1. Install Git, Node.js (use current LTS, e.g. 20+), and npm (comes with Node).

2. Auth is required via Auth0; values are currently hardcoded in Auth0Provider.tsx:18-20 and logout return URL in LogoutButton.tsx:11.

3. In Auth0 app settings, allow http://localhost:5173/callback and http://localhost:5173 (matching those files).

## Installation

Clone the repository:
git clone https://github.com/Heather-Rutherford/relational-databases-with-sqlalchemy.git

## After Cloning

1. Run npm install

2. Run npm run dev

3. Open http://localhost:5173 in a browser
